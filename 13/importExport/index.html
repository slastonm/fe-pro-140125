CommonJS (require) and ES6 (import/export) are two different ways to include external modules in your code.

CommonJS (require)

const myModule = require('./myModule');

ES Modules (import)

import вимагає, щоб файл мав розширення .mjs або щоб у package.json було вказано "type": "module"

import myModule from './myModule.js'; або import { myFunction } from './myModule.js
